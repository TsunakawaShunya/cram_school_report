# 涌井さん成果物
## 概要
現在は社員でのみログインができ，社員が講師(バイト)のアカウントを発行して，講師はログインできるようにしようと考えています．
また，現在はテスト用で社員が授業報告の作成も閲覧もできます．
生徒は社員が手動で追加するようになっており，その生徒について授業報告を書くことができます．

講師用と社員用でページは分けられていません．
